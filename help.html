General Information

  • Oria is an AI assistant designed to help you with knowledge-based tasks. It's created by Xexis and focuses on providing accurate, helpful information while maintaining professional boundaries. Oria can answer questions on a wide range of topics, explain complex concepts, and assist with various informational needs.

  • Oria was created by Xexis, a company founded by Bappi Bhadro. The team at Xexis developed Oria as an AI assistant focused specifically on knowledge and learning assistance. Unlike other AI models trained by different companies, Oria has been developed with a unique approach to provide accurate and helpful information.

  • Oria XSS 4Q is the current version of Oria. This version features enhanced knowledge processing capabilities, advanced reasoning, and improved contextual understanding. It's designed to provide more accurate and helpful information than previous versions. The "XSS 4Q" designation refers to the model architecture and version number in Xexis's development sequence.

  • Yes, Oria is currently free to use. You can access all of Oria's knowledge capabilities without any subscription or payment. However, creating an account is recommended to save your chat history and personalize your experience. Future premium features may be introduced, but the core functionality will remain accessible to all users.

  • Oria is a web-based application that works on a variety of devices including:

    • Desktop computers (Windows, Mac, Linux)
    • Smartphones (iOS, Android)
    • Tablets

    You can access Oria through most modern web browsers like Chrome, Safari, Firefox, and Edge. The interface is responsive and adapts to different screen sizes for optimal user experience.

  • Oria differentiates itself from other AI assistants in several key ways:

    • Knowledge Focus: Oria is specifically designed for knowledge sharing and educational purposes
    • Unique Training: Developed by Xexis with its own approach to AI learning
    • Clean Interface: Simplified design focused on clear information delivery
    • Privacy-Oriented: Conversations are stored locally on your device
    • Specialized Features: Tools like the country selector with map integration

    Unlike some other assistants that try to be all-purpose tools, Oria focuses on doing one thing exceptionally well: providing clear, accurate knowledge.

  • Yes, Oria requires an active internet connection to function. This is because:

    • The AI model that powers Oria's responses runs on cloud servers
    • Features like the country information and map display require API access
    • Account authentication and verification need internet connectivity

    If your internet connection is interrupted, Oria will attempt to reconnect automatically when the connection is restored. However, you'll still be able to view previously loaded chat history while offline.

  • Currently, Oria primarily supports English for optimal performance and accuracy. However, Oria can understand and respond to queries in several other languages, including:

    • Bengali
    • English
    • Hindi
    • Korean
    • Japanese
    • French
    • German
    • Portuguese
    • Italian

    Support for additional languages is continuously being improved. For the best experience, we recommend using English when possible, especially for complex or technical questions.

  • Advanced Mode (now called DeepX) is a special interaction mode in Oria that offers:

    • Short, precise, detailed responses
    • Responses focused specifically on what you ask
    • Professional knowledge delivery
    • Clear, focused communication
    • Intelligent learning support

    DeepX mode can be manually activated by toggling the DeepX switch in the tools popup. When active, you'll notice the chat subtitle and input placeholder change to indicate DeepX is enabled. This mode also unlocks URL analysis and social media video analysis capabilities.

    For more details on using DeepX mode, see the "What is DeepX mode and how do I use it?" question in this help section.

  • Oria is available on multiple platforms to meet your needs:

    • Web App: The full-featured web application accessible from any browser at oria.ai
    • Mobile App: Native applications for iOS and Android with enhanced mobile features

    The mobile apps offer additional features like:

    • System-level integrations with your device
    • Push notifications for important updates
    • Better performance and battery optimization

    If you prefer not to install the app, you can still access the web app through your mobile browser and add it to your home screen:

    For iOS: In Safari, tap the share icon and select "Add to Home Screen"

    For Android: In Chrome, tap the menu (three dots) and select "Add to Home Screen"

Features

  • Oria can assist you with a wide range of knowledge-based tasks, including:

    • Answering questions on various topics
    • Explaining complex concepts in simple terms
    • Providing information about countries and locations
    • Giving summaries on historical events, scientific concepts, and more
    • Offering educational content for learning purposes
    • Creating structured information on topics you're interested in

  • Oria's Location feature allows you to get information about countries with an integrated map display. Here's how to use it:

    1. Click the plus (+) button next to the input field
    2. Select the "Location" option
    3. In the country selector popup, you can:
      • Search for a specific country
      • Browse countries alphabetically
      • Use the alphabet navigation at the top
    4. Click on a country to see its location on a map
    5. Click "Select Country" to ask Oria about that country

    The feature uses OpenStreetMap to display country locations without requiring an API key, ensuring reliability and privacy.

  • You can upload images to Oria for analysis or reference. Here's how:

    1. Click the plus (+) button next to the input field
    2. Select "Image" from the tools menu
    3. Choose an image from your device (max size: 5MB)
    4. Once uploaded, you'll see a preview of the image
    5. Type your question about the image (optional)
    6. Click the send button

    Oria will analyze the image and respond with relevant information. This feature is useful for identifying objects, analyzing charts/graphs, or getting information about what's shown in the image.

  • Oria automatically saves your conversation history locally on your device. Additionally, you can save specific messages for easy access later:

    1. Hover over any message from Oria to reveal the action buttons
    2. Click the bookmark icon (save button)
    3. The message and your previous query will be saved
    4. Access saved messages by clicking on the bookmark icon in the sidebar

    This feature is useful for keeping important information or responses that you want to reference later. Saved messages are stored in the "Saved" section of the app.

  • There are multiple ways to start a new chat in Oria:

    • Method 1: Click the pen icon (new chat button) in the top right corner of the header
    • Method 2: Open the sidebar by clicking the menu button and then click "New Chat"
    • Method 3: If you're in an existing conversation, finish it first by sending at least one message

    Starting a new chat will clear the current conversation and create a fresh chat session. Don't worry - your previous conversation will be saved in your chat history and can be accessed from the sidebar.

  • Suggestion buttons are pre-defined topics or questions that appear at the bottom of the screen when you haven't started a conversation yet. They help you:

    • Get ideas for what to ask Oria
    • Quickly start conversations on common topics
    • Discover Oria's capabilities

    To use them, simply click on any suggestion button that interests you. Oria will automatically use that topic as your first message and respond accordingly. The suggestions change daily to offer fresh ideas and cover different topics.

  • You can easily share Oria's responses with others:

    1. Hover over the message you want to share
    2. Click the share icon (first button that appears)
    3. In the share popup, choose from several options:
      • Facebook
      • WhatsApp
      • Messenger
      • Email
      • More (uses your device's native sharing options)

    Alternatively, you can copy the text by clicking the copy icon (second button) and then paste it wherever you want to share it.

  • Oria offers both dark and light themes to suit your preference:

    1. Click the settings icon in the top right corner
    2. In the Settings page, find the "Appearance" section
    3. Choose between "Dark" and "Light" themes
    4. Your selection will be applied immediately and saved for future sessions

    You can also customize colors further in the "Color Palette" section of Settings. This allows you to personalize various elements of the interface to your liking.

  • Oria supports several keyboard shortcuts to enhance your productivity:

    • Enter: Send message
    • Shift + Enter: Add a new line in your message
    • Up Arrow: Edit your last message (if no response yet)
    • Ctrl/Cmd + /: Focus on the input field
    • Esc: Close popups or dialogs
    • Ctrl/Cmd + N: Start a new chat

    These shortcuts work on desktop browsers. Mobile devices primarily use touch interactions instead of keyboard shortcuts.

  • Yes, you can rename your chat conversations for better organization:

    1. Open the sidebar by clicking the menu button
    2. Find the chat you want to rename in the history
    3. Click the pencil icon on that chat
    4. Enter a new title for the chat
    5. Press Enter or click the checkmark to save

    By default, chats are titled based on your first message. Renaming them helps you find specific conversations more easily, especially if you have many chats on similar topics.

  • Oria allows you to generate images directly in the chat using the /image command:

    1. Type /image followed by your description
    2. For example: /image a beautiful sunset over mountains
    3. Press Enter to send the command
    4. Oria will generate and display the image based on your description

    The generated images will include action buttons in the top-right corner that allow you to download the image or view it in full size. You can also like or dislike the image to provide feedback.

  • Similar to image generation, Oria allows you to generate videos using the /video command:

    1. Type /video followed by your description
    2. For example: /video waves crashing on a beach
    3. Press Enter to send the command
    4. Oria will find and display a video matching your description

    The videos will appear with playback controls, and you can like or dislike them to provide feedback. You can also download videos or view them in full screen using the action buttons in the top-right corner.

  • DeepX is an enhanced research mode that provides more detailed and thorough responses:

    1. Click the plus (+) button next to the input field to open the popup
    2. Look for the DeepX toggle switch in the popup
    3. Toggle the switch to enable DeepX mode

    When DeepX mode is enabled:

    • You'll notice the input placeholder changes to "Ask anything (DeepX mode)..."
    • The empty chat subtitle changes to indicate DeepX is active
    • Responses will be more detailed and thoroughly researched
    • URL and social media video analysis capabilities are unlocked

    DeepX mode is perfect for complex questions or when you need more comprehensive information. Your DeepX preference is saved between sessions.

  • When DeepX mode is enabled, Oria gains the ability to analyze URLs shared in the chat:

    • Share any URL in the chat while DeepX mode is active
    • Oria will extract key information from the webpage
    • For articles, you'll get summaries of the content
    • For product pages, you'll get details about the product
    • For news sites, you'll get context about the article

    This feature helps you quickly understand website content without having to visit the site or read through everything. It's especially useful for research, comparison shopping, or staying informed about current events.

  • When DeepX mode is enabled, Oria can analyze videos from social media platforms:

    1. Share a link to a video from platforms like YouTube, TikTok, Instagram, etc.
    2. Oria will analyze the video content and provide insights
    3. You'll receive information about the video's topic, key moments, and context

    This feature is useful for:

    • Understanding what a video is about without watching it
    • Finding specific information within a video
    • Getting context about viral or trending content
    • Fact-checking video claims

    Note that the analysis is based on available metadata and content recognition technology, so accuracy may vary depending on the video.

  • Oria allows you to provide feedback on generated images and videos through like/dislike buttons:

    1. Generate an image or video using the appropriate command
    2. Look for the message actions at the bottom of the content
    3. Click the thumbs-up icon to like the content
    4. Click the thumbs-down icon to dislike the content

    Your feedback is saved locally and helps personalize your experience. Liked content may be given preference in future generations, while disliked styles will be avoided. This feature uses localStorage to remember your preferences between sessions.

  • Yes, you can download any image or video generated in your chat:

    1. Hover over the image or video to reveal the action buttons in the top-right corner
    2. Click the download icon (down arrow)
    3. The file will be saved to your default downloads folder

    For videos, you can also:

    1. Right-click on the video and select "Save Video As..." from the context menu
    2. Choose a location on your device to save the video

    Downloaded content is for personal use only. Please respect copyright and usage rights for any generated content.

  • The /image command supports a wide variety of image types and styles:

    Image Types:

    • Landscapes and nature scenes
    • Abstract and conceptual imagery
    • Object illustrations
    • Scenes with multiple elements
    • Artistic interpretations

    Tips for better results:

    • Be specific in your descriptions
    • Include details about style, mood, lighting, and composition
    • Use adjectives to clarify the look you want
    • Try different prompts if the first result isn't what you expected

    For example, instead of just /image mountain, try /image snow-capped mountain at sunset with dramatic clouds and golden light.

  • The /video command can generate various types of videos:

    Video Types:

    • Nature scenes (beaches, forests, mountains, etc.)
    • Urban environments and cityscapes
    • Animal footage
    • Time-lapse sequences
    • Various activities and events

    Tips for better results:

    • Be specific about what you want to see
    • Include motion descriptions (e.g., "flowing water," "birds flying")
    • Specify the setting or environment
    • Mention if you want slow-motion or time-lapse

    For example, instead of just /video city, try /video busy Tokyo intersection at night with glowing neon signs and people crossing.

  • While Oria strives to provide unlimited access to features, there may be some practical limitations:

    • Image generation: Currently limited to 50 images per day per user
    • Video generation: Currently limited to 25 videos per day per user
    • Image size: Generated images are provided in standard resolution
    • Video length: Generated videos are typically 10-30 seconds in length

    These limits help ensure fair usage and system stability. Premium accounts may have higher limits in the future. If you reach your daily limit, you'll receive a message indicating that you've used all your generations for the day, and the limit will reset at midnight UTC.

Account

  • To create an Oria account:

    1. Click on the user icon in the top right corner
    2. Select "Sign Up" in the authentication dialog
    3. Add a profile picture (required)
    4. Enter your username
    5. Enter your Gmail address (only Gmail is currently supported)
    6. Create a secure password
    7. Click "Sign Up"
    8. Check your email for a verification code and enter it when prompted

    After verification, you can sign in with your new account.

  • While you can use some basic features of Oria without an account, creating one provides several benefits:

    • Persistent chat history across devices
    • Access to saved messages and favorites
    • Personalized experience based on your preferences
    • Ability to customize settings and themes
    • Enhanced security for your conversations
    • Access to future premium features

    An account ensures that your data and preferences are saved securely and can be accessed whenever you use Oria, regardless of the device you're using.

  • Currently, Oria only supports Gmail accounts (@gmail.com) for user registration for several reasons:

    • Reliable email verification process
    • Enhanced security features
    • Simplified authentication system during beta phase
    • Wider compatibility with planned future features

    Support for additional email providers is planned for future updates. If you don't have a Gmail account, you can create one for free at mail.google.com to use with Oria.

  • If you've forgotten your password, follow these steps to reset it:

    1. Navigate to the Settings page by clicking the settings icon
    2. Scroll down to the "Account" section
    3. Find "Forgot Password?" and click on "Contact Support"
    4. This will open your default email client with a pre-filled email
    5. Complete any additional information and send the email
    6. The support team will verify your identity and help reset your password

    For security reasons, password resets are handled manually by our support team to ensure that only legitimate account owners can reset passwords.

  • To edit your profile information:

    1. Click on your profile picture in the top right corner
    2. Select "Edit Profile" from the dropdown menu
    3. In the Edit Profile dialog, you can:
      • Change your profile picture by clicking on the current image
      • Update your username
    4. Click "Save Changes" to update your profile

    Note that your email address cannot be changed directly. If you need to change your email, please contact support through the Settings page.

  • To log out of your Oria account:

    1. Click on your profile picture in the top right corner
    2. Select "Sign Out" from the dropdown menu
    3. Confirm that you want to sign out when prompted

    After signing out, you'll return to the login screen. For security reasons, especially on shared or public devices, always remember to log out when you're finished using Oria. If you're on your personal device, you can stay logged in for convenience.

  • Oria requires a profile picture for several reasons:

    • Enhanced user experience with visual identification
    • Better personalization of the interface
    • Easier account recognition in multi-user environments
    • Reduced likelihood of automated account creation

    You can use any appropriate image as your profile picture - it could be a photo of yourself, an avatar, or any image that represents you. The image is stored securely and is only used within the Oria application.

  • While there's no strict limit on the number of accounts you can create, we recommend maintaining a single account for the best experience. Each Gmail address can be associated with one Oria account.

    Creating multiple accounts unnecessarily may lead to:

    • Fragmented chat history across accounts
    • Complications managing saved content
    • Potential confusion with settings and preferences

    If you have a specific need for multiple accounts, please use different email addresses for each account to avoid conflicts.

  • To delete your Oria account:

    1. Go to the Settings page by clicking the settings icon
    2. Scroll to the Account section
    3. Click on "Contact Support" in the Account section
    4. In the pre-filled email template, change the subject to "Account Deletion Request"
    5. Provide a brief explanation for why you're deleting your account
    6. Send the email from your registered email address

    Our support team will process your request within 48 hours. Account deletion is permanent and will remove all your data, including chat history and saved messages. This action cannot be undone.

  • Oria accounts and associated data are maintained even during periods of inactivity. If you don't use your account for an extended period:

    • Your account remains active and accessible
    • Your chat history and saved messages are preserved
    • Your settings and preferences remain unchanged

    There is currently no automatic deletion of inactive accounts. However, we may implement an inactivity policy in the future, in which case we would notify all users via email before any data is affected. If you plan to stop using Oria permanently, we recommend properly deleting your account as described in the account deletion FAQ.

Privacy

  • Oria takes your privacy seriously. Here's how your data is handled:

    • Chat history is stored locally on your device
    • Your conversations are not used to train the AI model
    • Account information is stored securely and used only for authentication
    • We do not sell or share your personal information with third parties
    • You can delete chat history at any time

  • Yes, your conversations with Oria are private. We maintain privacy through several measures:

    • Conversations are stored locally on your device using browser localStorage
    • Messages are not accessible to other users or Xexis employees
    • Your queries are processed by the AI system but not stored permanently on our servers
    • We don't use your conversations to train or improve the AI model
    • Data in transit is protected using modern encryption standards

    While Oria needs to process your messages to generate responses, this processing is temporary and focused on providing you with accurate answers, not on collecting or analyzing your data.

  • Yes, Oria uses a minimal set of cookies and localStorage for essential functionality:

    • Authentication cookies: To keep you logged in
    • Preference cookies: To remember your theme and settings
    • localStorage: For storing chat history and saved messages

    These are used only for improving your user experience and maintaining your preferences. We do not use third-party tracking cookies or advertising cookies. You can clear browser storage and cookies at any time, though this will reset your preferences and logged-in state.

  • Yes, you can delete your chat history in several ways:

    To delete a specific chat:

    1. Open the sidebar by clicking the menu button
    2. Find the chat you want to delete
    3. Click the trash icon next to that chat
    4. Confirm deletion when prompted

    To clear all chat history:

    1. Go to Settings by clicking the settings icon
    2. Scroll down to the "Privacy" section
    3. Click "Clear All Chat History"
    4. Confirm your choice when prompted

    Alternatively, you can clear site data through your browser settings, which will remove all stored data including chat history and preferences.

  • No, Oria does not have access to your device files or system unless you explicitly grant permission for specific actions:

    • When you upload an image, you select the specific file to share
    • When using location features, you explicitly grant location access
    • Oria runs in your browser and has the same limitations as any website

    Oria cannot:

    • Access your device's file system without your permission
    • Read or modify files on your device
    • Access your camera or microphone without explicit permission
    • Run in the background when you're not using the application

    All permissions are requested clearly and can be revoked at any time through your browser settings.

  • Oria only accesses your location when you explicitly use the location-related features:

    • When you click "Use Current Location" in the country selector
    • When you specifically ask Oria about your current location

    In these cases:

    • Your browser will ask for permission before sharing location
    • You can deny location access and still use all other features
    • Location data is used only for the specific request and not stored
    • Precise coordinates are converted to general area information

    Oria does not track your location in the background or use location data for any purpose other than fulfilling your specific request.

  • While Oria maintains high security standards, we recommend exercising caution when sharing sensitive personal information:

    Information we recommend NOT sharing:

    • Passwords or authentication credentials
    • Financial information (credit card numbers, banking details)
    • Social security numbers or government identification
    • Private health information
    • Home addresses or detailed personal schedules

    It's generally safe to share general information like your interests, general location (city/country), or non-sensitive questions and topics. Remember that your conversations are stored locally on your device, so anyone with access to your device could potentially view your chat history.

  • Oria implements several security measures to protect your account:

    • Email verification: All accounts require email verification
    • Password security: Passwords are hashed, not stored in plain text
    • Secure connections: All data is transmitted over HTTPS
    • Limited login attempts: Protection against brute force attacks
    • Manual password reset: Prevents unauthorized account access

    To further enhance your account security:

    • Use a strong, unique password
    • Don't share your login credentials
    • Sign out when using shared devices
    • Keep your email account secure (since it's linked to your Oria account)

  • Yes, Oria is designed to comply with major data protection regulations, including:

    • GDPR (General Data Protection Regulation): For users in the European Union
    • CCPA (California Consumer Privacy Act): For users in California
    • PIPEDA (Personal Information Protection and Electronic Documents Act): For users in Canada

    Our approach to compliance includes:

    • Minimizing data collection to only what's necessary
    • Providing transparency about data usage
    • Giving users control over their personal data
    • Implementing appropriate security measures
    • Facilitating data access and deletion requests

    If you have specific questions about data protection or wish to exercise your rights under these regulations, please contact our support team.

Troubleshooting

  • If Oria isn't responding, try these troubleshooting steps:

    1. Check your internet connection
    2. Refresh the page by pulling down from the top
    3. Clear your browser cache and cookies
    4. Try using a different browser
    5. If the problem persists, please contact support

  • If your chat history has disappeared, try these recovery steps:

    1. Check if you're logged in: Make sure you're logged into the correct account
    2. Clear browser data issues: If you recently cleared browser data, this might have removed locally stored chats
    3. Check sidebar: Open the sidebar to see if your chats are still listed but not visible in the main view
    4. Device check: If you're using a different device, remember that chat history is device-specific unless you're logged in
    5. Local storage check: Some privacy settings or browser extensions might restrict local storage access

    Unfortunately, if none of these steps work, the chat history may not be recoverable. To prevent future loss, consider:

    • Using the save feature for important information
    • Logging in to access chats across devices
    • Not using incognito/private browsing mode (which doesn't preserve localStorage)

  • If you're experiencing issues with the map feature, try these solutions:

    1. Check internet connection: Maps require a stable connection to load properly
    2. Allow iframe content: Make sure your browser isn't blocking iframe content (check security settings or extensions)
    3. Enable JavaScript: The map requires JavaScript to function correctly
    4. Try a different browser: Some browsers handle maps better than others
    5. Clear browser cache: Old cached data might be interfering with map loading

    For location-specific issues:

    • Make sure location access is allowed in your browser settings
    • Some VPNs or proxy services can affect location accuracy
    • Try selecting countries manually instead of using current location if problems persist

  • If you're having trouble uploading images, check these common causes:

    1. File size too large: Oria has a 5MB size limit for images
    2. Unsupported file format: Make sure you're uploading common image formats (JPG, PNG, GIF, etc.)
    3. Browser permissions: Your browser needs file access permission
    4. Storage issues: Your device might be low on storage space
    5. Network issues: Poor connection can interrupt uploads

    Try these solutions:

    • Compress or resize the image to under 5MB
    • Convert to a standard format like JPG or PNG
    • Try a different image to see if the issue is file-specific
    • Use a stable internet connection
    • Refresh the page and try again

  • While Oria strives for accuracy, it may occasionally provide incorrect information. If this happens:

    1. Try rephrasing your question: Sometimes a clearer or more specific question helps
    2. Provide more context: Additional details can help Oria understand what you're asking
    3. Check for recent developments: Oria's knowledge may not include very recent events or changes
    4. Verify from other sources: For critical information, always verify with authoritative sources

    If you notice a consistent pattern of incorrect information on a particular topic, please report it to help improve Oria's accuracy. You can do this by:

    • Sending feedback through the Settings page
    • Providing specific examples of the incorrect information
    • Suggesting the correct information with sources if possible

  • If Oria is running slowly, try these performance optimization tips:

    1. Clear chat history: Long conversation histories can slow down the app
    2. Close other browser tabs/apps: These compete for system resources
    3. Clear browser cache: Accumulated cache can affect performance
    4. Check your internet connection: Slow connections affect response times
    5. Use a modern browser: Older browsers may not handle the app efficiently

    Additional tips for mobile devices:

    • Close background apps to free up memory
    • Ensure your device has sufficient storage space available
    • Use the "Add to Home Screen" option for better performance
    • If possible, connect to WiFi instead of using mobile data

  • If you're experiencing login issues, try these troubleshooting steps:

    1. Check your credentials: Ensure you're using the correct email and password
    2. Verify email verification: Make sure you've completed the email verification process
    3. Clear cookies and cache: Corrupted cookies can interfere with login
    4. Disable extensions: Some browser extensions can block login processes
    5. Try incognito mode: This can help identify if browser settings are causing issues
    6. Check caps lock: Ensure caps lock isn't accidentally on

    If you still can't log in:

    • Use the "Forgot Password" option in Settings to reset your password
    • Try accessing from a different device or browser
    • Check if your email address is typed correctly (only Gmail addresses are supported)
    • Contact support if problems persist

  • If the theme or colors don't display correctly, try these solutions:

    1. Reset theme settings: Go to Settings > Appearance and select either Light or Dark theme
    2. Reset color palette: In Settings > Color Palette, click "Reset to Default Colors"
    3. Clear localStorage: This will reset all settings but can fix corrupted theme data
    4. Check browser compatibility: Make sure you're using a supported modern browser
    5. Disable color management extensions: Browser extensions that modify colors can interfere

    Device-specific issues:

    • Dark mode integration: Some devices have system-wide dark mode that can interfere
    • Accessibility settings: High contrast or color filters can affect appearance
    • Display settings: Night mode, blue light filters, or custom color profiles can affect colors

    If issues persist, try accessing Oria from a different device to see if the problem is device-specific.

  • If you haven't received your verification email, follow these steps:

    1. Check spam/junk folders: Verification emails sometimes get filtered
    2. Wait a few minutes: Email delivery can sometimes be delayed
    3. Verify email address: Double-check that you entered your email correctly
    4. Check Gmail categories: Look in Promotions or Updates tabs if using Gmail
    5. Check Gmail filters: Make sure you don't have filters automatically archiving these emails

    If you still don't receive the email after 15 minutes:

    • Try signing up again with the same email
    • Make sure your inbox isn't full
    • Ensure your email provider isn't blocking emails from our domain
    • Try using a different Gmail address if possible

    If problems persist, please contact support for assistance.
